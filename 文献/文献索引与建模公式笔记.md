## 4. 建模最常用公式

### 4.1 曝气池 DO 动态质量平衡

来自 2024 年曝气控制综述，适合写在参赛报告“机理基础”部分：

```text
dSo(t)/dt = [Qin(t) * Soin(t) - Qout(t) * So(t)] / V(t)
            + kLa(Qair(t)) * (So,sat - So(t))
            - So(t) / (Ko + So(t)) * Rr(t)
```

可用于说明：
1. 进水流量和进水 DO 会影响池内 DO。
2. 曝气量通过 `kLa` 影响氧传递。
3. 微生物呼吸耗氧使 DO 持续下降。

### 4.2 氧传递速率 OTR 公式

```text
dC/dt = KLa * (Cs - C)
```

可用于说明：
1. 曝气效率的核心不只是风量，还包括氧传递系数 `KLa`。
2. `KLa` 是曝气系统设计、控制和节能分析中的关键参数。

### 4.3 Fick 扩散定律到氧传递公式

```text
dM/dt = -D * A * dC/dx
```

可用于说明氧从气相进入液相的传质基础。

### 4.4 机理模型的一般质量平衡思路

ASM1 文章与 GPS-X 文章共同支持如下思路：

```text
状态变化 = 流入 - 流出 + 生化反应速率
```

可在报告中写成：
1. 进水负荷与流量决定输入项。
2. 池容和回流决定停留时间与稀释效应。
3. 生物反应速率由微生物动力学参数决定。

### 4.5 IFFNN 文章中的离散递推表达

```text
Ci+1 = Ci + [W(t)/V - λ * Ci] * (ti+1 - ti)
```

含义：
1. 未来出水浓度不仅与当前负荷相关。
2. 还与当前出水状态本身相关。
3. 非常适合支撑“当前状态 + 历史输出 + 当前控制量”的监督学习输入设计。

### 4.6 线性插值补全

CNN-LSTM 文章给出的缺失补全形式：

```text
Y = Y1 + (X - X1) * (Y2 - Y1) / (X2 - X1)
```

适合用于：
1. 监测数据短缺口补全。
2. 比赛中解释缺失值修复方法。

### 4.7 多目标优化目标函数

适合本项目自行写成：

```text
Min J = w1 * 出水超标风险 + w2 * 曝气能耗 + w3 * 药剂消耗 + w4 * 控制波动惩罚
```

该表达可由 LSTM-MPC、经济 MPC、智能加药文章共同支撑。

## 6. 对本项目最值得直接复用的设计结论

1. 进水参数不能只看 COD 和流量，至少还应加入 NH3-N、TP、TN、pH、SS 和温度。
2. 过程状态不能只看 DO，还应引入 MLSS、ORP、NO3-N，才能更稳定地反映工况。
3. 曝气控制的核心不是单纯提高风量，而是围绕 `KLa`、DO 和氧需求做动态平衡。
4. 加药控制不能只按经验线性放大，应把 TP、流量、pH 和历史出水效果一起考虑。
5. 监督学习输入应包含当前出水和历史时序特征，否则未来出水预测会明显变差。
6. 决策优化不能直接输出“最大去除率”方案，而要加入能耗、药耗和调节幅度惩罚。
7. 比赛答辩时最好强调“机理规则约束 + 数据驱动预测 + 优化决策”的组合，而不是单一黑箱模型。
